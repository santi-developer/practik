Describe que hacen los siguientes comandos de git, escribe su descripcion al frente en una sola linea.

git status: Este comando se usa para obtener el estado actual del repositorio.

git clone: Este comando se usa para obtener una copia local de un repositorio git remoto.  

git pull: Este comando se usa para actualizar el repositorio local con los últimos cambios del repositorio remoto.

git checkout: Este comando se usa para cambiar (navegar) entre ramas o versiones de archivos. 

git log: Este comando se usa para mostrar el historial de commits de un repositorio.

git branch: Este comando se usa para administrar y trabajar con ramas en un repositorio. 

git add: Este comando se usa para agregar cambios realizados en archivos al área de preparación (staging área).

git commit: Este comando se usa para crear un nuevo commit que registra y guarda los cambios realizados en el repositorio. 

git push: Este comando se usa para enviar los commits realizados en tu repositorio local a un repositorio remoto.  

git merge: Este comando se usa para combinar los cambios de una rama en otra.
